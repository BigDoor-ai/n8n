## 🧩 How to Use These Workflows

1. Import the `.json` file into your n8n instance  
2. Configure required credentials:
   - OpenAI  
   - Google Sheets  
   - Any third-party APIs  
3. Replace placeholder values (API keys, IDs, etc.)  
4. Activate the workflow  

---

## 🔒 Security Notice

- No sensitive credentials are included in this repository  
- All API keys and tokens must be added manually in your n8n environment  
- Always use environment variables or credential managers  

If you upload your own workflows, **sanitize them first** unless you enjoy public data leaks.
